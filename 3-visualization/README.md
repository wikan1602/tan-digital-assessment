# Visualization Assessment: Indonesia Cooperative Dashboard

## Overview
This dashboard analyzes the cooperative landscape in Indonesia using mock data. It highlights the distribution of cooperative types, membership demographics, and regional density.

## Technology Stack
- **Tool**: Microsoft Power BI Desktop
- **Data Source**: Python-generated mock data (simulating http://nik.depkop.go.id/)

## Key Insights
1. **Geospatial Distribution**: Map visualization showing cooperative density across provinces.
2. **Demographics**: Comparison of Male vs Female active members.
3. **Sector Analysis**: Breakdown of cooperative types (Simpan Pinjam, Produsen, etc.).
4. **Top Regions**: Top 5 provinces with the highest number of cooperatives.
